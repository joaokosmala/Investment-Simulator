# 📈 Investment Simulator

Um simulador de investimentos moderno desenvolvido com HTML, CSS e JavaScript, projetado para calcular rendimentos com juros compostos, aportes mensais e projeções financeiras de longo prazo.

Veja em: [Clique aqui](https://investmentsimulatortest.netlify.app/)

O projeto fornece uma visualização clara da evolução do patrimônio ao longo do tempo através de gráficos interativos, permitindo que usuários simulem diferentes cenários de investimento de forma rápida e intuitiva.

---

## 🚀 Funcionalidades

### 💰 Simulação de Investimentos

* Capital inicial personalizado.
* Aportes mensais configuráveis.
* Taxa de juros mensal ajustável.
* Prazo de investimento em meses.

### 📊 Cálculos Financeiros

* Juros compostos.
* Valor total investido.
* Juros acumulados.
* Montante final projetado.

### 📈 Visualização de Crescimento

* Gráfico interativo utilizando Chart.js.
* Evolução do patrimônio mês a mês.
* Comparação visual do crescimento do investimento.

### 📱 Interface Responsiva

* Compatível com desktops, tablets e smartphones.
* Layout moderno e intuitivo.
* Experiência otimizada para diferentes dispositivos.

## 📈 Exemplo de Simulação

### Entrada

```text
Capital Inicial: R$ 10.000
Aporte Mensal: R$ 500
Taxa de Juros: 1% ao mês
Prazo: 24 meses
```

### Resultado

```text
Valor Investido: R$ 22.000,00
Juros Obtidos: R$ 3.581,92
Montante Final: R$ 25.581,92
```
